详解EXIF-数码相机可交换的图像文件格式


缩写 English 中文
DSC Digital still camera 数码相机
DVC Digital video camera 数码摄像机
DTV Digital television 数字电视
Primary image The main image data 主要图像数据
Thumbnail A small image used to index the primary image 用于索引主图像的小图像
Compressed data Data referred to in the JPEG Standard as “entropy coded data” JPEG标准中称为“熵编码数据”的数据
Tag A field recording ancillary data about an image. Corresponds to “field” in TIFF 记录有关图像的辅助数据的字段。对应于TIFF中的“字段”
Tag information A description of the tag contents 标签内容的说明
Exif An abbreviation of “Exchange image file format,” used here as the general term for this standard and earlier versions of this standard “ Exchange映像文件格式”的缩写，在此用作本标准和本标准早期版本的总称
DCF An abbreviation of “Design rule for Camera File System”; it defines folder structures based on Exif-JPEG and formats for recording in memory, and is the camera file system standard that guarantees interoperability among devices of different types. 根据DCF记录的可移动内存，或 文件系统可以移动的和不可移动的内存通过IF从外部设备进行访问（无论有线或有线无线）
DCF media Removable memory recorded in compliance with the DCF or, removable and non-removable memory that a file system can access from an external device through IF (regardless of wired or wireless) 根据DCF记录的可移动内存，或 文件系统可以移动的和不可移动的内存通过IF从外部设备进行访问（无论有线或有线无线）
Exif/DCF Writer A digital still camera (DSC) or other device or application software that can output an Exif/DCF file 数码相机（DSC）或其他可以输出一个Exif / DCF文件
Exif/DCF Editor A device or application that edits (i.e., makes partial additions, changes, deletions) existing Exif/DCF files (Alternatively, image-editing application; includes some DTV, photo frames, and DSC) 进行编辑（即进行部分添加，更改，删除）现有的Exif / DCF文件 （或者，图像编辑应用程序；包括一些DTV，相框，和DSC）
Exif/DCF Reader A device, such as a printer with a memory card slot, capable of directly reading an Exif/DCF file and printing or performing other output processing, as well as application software that uses Exif tags to perform image processing for output. (with playback function in conformity with the DCF Reader 1 and Reader 2 specifications described in 2.1 “Definition of Terms” of the DCF Standard) 一种设备，例如带有存储卡插槽的打印机，可以直接读取Exif / DCF文件并打印或执行其他输出处理，以及使用Exif标签执行图像的应用程序软件处理输出。（具有符合DCF的播放功能阅读器1和阅读器2的规范在2.1“术语的定义”中进行了描述DCF标准）
